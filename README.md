# weightIn

Is a javascript module you can use for convert kilograms to pounds

## Description
- Convert number as number or string
- Words errors

## install
```
npm install weight-in
```

## Use
```
import weightIn from 'weight-in\'

//To kilograms
weightIn.kilograms(1)//0.45

//To pounds
weightIn.pounds(1)//2.204
```
